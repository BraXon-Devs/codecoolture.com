# React: Using children instead of dedicated slots

Using component props to pass data (or `JSX` elements directly) that will later
populate inner components has been a prevalent pattern in the React community
trying to emulate [the better-designed slot approach adopted by Vue][vue-slots].

For example, imagine we are coding a new `Nav` component for our site. The `Nav`
will render a collection of nav items, the page title, and a call to action
link. Each page might customize all these elements.

Not surprisingly, I think either one of these two would be the most popular
examples we might find out there (or even a combination of both):

```tsx
// Passing simple data structures with the desired data
<Nav
  cta={{ text: "Buy the book!", href: "https://url.to/buy-my-book" }}
  menuItems={[
    { text: "Home", href: "/home" },
    { text: "About", href: "/about" },
    { text: "Contact", href: "/contact" },
  ]}
  title="My Site"
/>

// Passing JSX elements directly
<Nav
  cta={<a href="https://url.to/buy-my-book">Buy the book!</a>}
  menuItems={
    <>
      <li>
        <a href="/home">Home</a>
      </li>

      <li>
        <a href="/about">About</a>
      </li>

      <li>
        <a href="/contact">Contact</a>
      </li>
    </>
  }
  title={<h1>My Site</h1>}
/>
```

[continue talking here about the problems with this approach]

## Possible solution

```tsx
<Nav>
  <Nav.Title>My Site</Nav.Title>

  <Nav.Menu>
    <Nav.Menu.Item href="/">Home</Nav.Menu.Item>
    <Nav.Menu.Item href="/about">About</Nav.Menu.Item>
    <Nav.Menu.Item href="/contact">Contact</Nav.Menu.Item>
  </Nav.Menu>

  <Nav.CallToAction href="https://url.to/buy-my-book">
    Buy the book!
  </Nav.CallToAction>
</Nav>
```

---

## Possible solution

```tsx
type NavProps = { children: React.ReactNode };

const Nav = ({ children }: NavProps) => {
  return <nav>{children}</nav>;
};

type TitleProps = { children: React.ReactNode };

const Title = ({ children }: TitleProps) => {
  return <h1>{children}</h1>;
};

type MenuProps = { children: React.ReactNode };

const Menu = ({ children }: MenuProps) => {
  return <ul>{children}</ul>;
};

type MenuItemProps = { children: React.ReactNode; href: string };

Menu.Item = ({ children, href }: MenuItemProps) => {
  return (
    <li>
      <a href={href}>{children}</a>
    </li>
  );
};

type CallToActionProps = { children: React.ReactNode; href: string };

const CallToAction = ({ children, href }: CallToActionProps) => {
  return <a href={href}>{children}</a>;
};

Nav.Title = Title;
Nav.Menu = Menu;
Nav.CallToAction = CallToAction;

export { Nav };
```

That will be used as the following:

```tsx
<Nav>
  <Nav.Title>My Site</Nav.Title>

  <Nav.Menu>
    <Nav.Menu.Item href="/">Home</Nav.Menu.Item>
    <Nav.Menu.Item href="/about">About</Nav.Menu.Item>
    <Nav.Menu.Item href="/contact">Contact</Nav.Menu.Item>
  </Nav.Menu>

  <Nav.CallToAction href="https://url.to/buy-my-book">
    Buy the book!
  </Nav.CallToAction>
</Nav>
```
